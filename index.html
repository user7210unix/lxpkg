The independent Package Manager for Gnu/Linux Distributions

In collaboration with the LearnixOS Team.

Features 📋

  • Developed from Scratch ⚙️
  • Written in Python 🐍
  • Lightweight and Fast ⚡
  • Uses TOML Configuration Files 📄
  • Custom Repository Support 🗃️

Commands ⌨️

  • sudo lxpkg install
  • sudo lxpkg list
  • sudo lxpkg search
  • sudo lxpkg remove

INSTALLATION 🍃

curl -fsSL https://raw.githubusercontent.com/user7210unix/lxpkg/main/install.sh | bash

Copy this into your Linux Terminal
